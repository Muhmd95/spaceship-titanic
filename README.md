# Spaceship Titanic: Kaggle Competition Solution

This repository contains my solution for the [Spaceship Titanic](https://www.kaggle.com/competitions/spaceship-titanic) Kaggle competition. The goal of this project is to predict which passengers were transported to an alternate dimension during the Spaceship Titanic's collision with a spacetime anomaly.
